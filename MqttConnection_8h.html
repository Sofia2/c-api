Deprecated:

Initializes a MQTT connection.

Parameters
clientIdThe MQTT clientId to be used.
Note
The memory of the clientId will NOT be freed after the connection is closed.
Parameters
credentialsThe MQTT credentials to be used.
messageReceivedCallbackA pointer to the function that will process all the incoming SSAP messages (except the INDICATION ones)
messageReceivedContextA pointer to the context (a.k.a. "state") structure used by the messageReceivedCallback function.
connectionEventsCallbackA pointer to the function that will process the MQTT connection events. So far, only disconnection events are detected.
connectionEventsContextA pointer to the context (a.k.a. "state") structure used by the connectionEventsCallback function.
Note
The memory of the connection and of all the given parameters will be freed when the connection is destroyed.
